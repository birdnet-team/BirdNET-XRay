# BirdNET-XRay
BirdNET sound ID demo

# Setup
```
pip install tensorflow screeninfo librosa
sudo apt-get install ffmpeg python3-pyaudio python3-opencv
```

# Run
```
python demo.py
```

Keyboard shortcuts: 

- `esc` to quit
- `p` to pause/resume
- `s` to save the current frame
- `c` to change the colormap
- `a` to play next soundscapes
